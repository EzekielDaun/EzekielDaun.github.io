笔记:

  • 自动更新 auto_reload
    pelican -r
  • 重新生成时清空 output 文件夹,保留指定文件夹
    DELETE_OUTPUT_DIRECTORY = TRUE
    OUTPUT_RETENTION = [".hg", ".git", ".bzr"]
  • 文件名正则提取 metadata title & date
    FILENAME_METADATA = '(?P<title>.*)(?P<date>\d{4}\d{2}\d{2})'
  • GitHub 更新 css 文件可能较慢? 改变 theme 后不生效可能需要多等等
  • Pymdownx extensions
    • Details
      • ???+ 为默认打开,不带+默认关闭
    • critics
      • {==mark 中文==}
      • {--del--}
      • {++ins++}
  • 使用 <br> 换行以避免回车换行产生 <p> 标签
  • mdx_include --以 {! xxx.xxx !} 的形式在 md 中添加文件
    pip install mdx_include 'mdx_include':{'base_path': 'path/to/your/directory',}
    使用 \ 来取消解析
  • svg 自动跟随 body 颜色(仅限黑白图片)
    画 svg 的时候设置 strokeColor=currentColor ,其他各种 color 也都要设为 currentColor
  • 深色模式平滑切换
    使用 rel="alternate stylesheet" 来预载全部css避免闪烁,切换时设置 rel="stylesheet" 即可
